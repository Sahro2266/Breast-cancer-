Breast Cancer Classification

A machine learning project that applies multiple classification algorithms to predict whether a breast cancer tumor is malignant or benign using the Wisconsin Breast Cancer dataset.

Overview

This project compares the performance of five machine learning models on a real-world medical dataset to identify the best-performing classifier for breast cancer diagnosis.

Models Used


K-Nearest Neighbors (KNN)
Support Vector Machine (SVM)
Decision Tree
Random Forest
XGBoost


Tools & Libraries


Python
scikit-learn
XGBoost
Pandas
NumPy
Matplotlib / Seaborn
Google Colab


Workflow


Loaded and explored the Wisconsin Breast Cancer dataset
Preprocessed data using StandardScaler for feature normalization
Applied feature selection to identify the most relevant predictors
Trained and evaluated five classification models
Compared model performance using accuracy, precision, recall, and F1-score


Results

Multiple classifiers were trained and evaluated. Random Forest and XGBoost achieved the highest accuracy, demonstrating strong performance on this binary classification task.

How to Run

Open the notebook in Google Colab or Jupyter Notebook and run all cells in order. No additional setup is required — all dependencies are standard and available in Colab.
